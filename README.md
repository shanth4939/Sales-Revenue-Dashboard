# Sales-Revenue-Dashboard
This project is a Power BI dashboard developed for analyzing sales and revenue performance using the Superstore dataset.
# Features
- Total Sales KPI
- Total Profit KPI
- Sales Trend Analysis
- Top Performing Products
- Region-wise Filtering

## Tools Used
- Power BI
- Superstore Dataset

## Project Outcome
This dashboard helps visualize business insights and track sales performance interactively.
